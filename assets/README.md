# Assets Folder

This folder contains static assets for the Gigfactory website.

## Structure

- `images/` - Image files (logos, icons, photos)
- `icons/` - Icon files and SVGs
- `fonts/` - Custom font files (if any)

## Usage

Place your static assets in the appropriate subfolders and reference them in your components using relative paths.

Example:
```javascript
import logo from '../assets/images/logo.png'
```
